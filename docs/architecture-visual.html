TokenLens

Universal AI token monitoring and prediction platform for coding tools. Track, forecast, and optimize your LLM token consumption.

System Architecture

Click any component to expand details

📡 Adapters

Claude Code (.jsonl parser) and Kiro (MCP + tiktoken estimation). Pluggable via entry-points. Synchronous I/O wrapped in asyncio.to_thread().

⚙️ Agent (Daemon)

Background daemon with watchdog file watcher, event pipeline (dedup + batch), session manager (gap detection), and ML task runner. PID file + heartbeat for health.

🧠 ML Pipeline

Forecaster (Holt-Winters/Prophet), Anomaly (IsolationForest), Efficiency (5-factor scoring), Profiler (KMeans), Budget (cost projection). Models persisted via joblib.

🚀 API Server

FastAPI with 9 route groups, WebSocket live push (5s), alert broadcast, Swagger docs at /docs. CORS configured for frontend dev server.

🔔 Alert Engine

Threshold triggers (50/75/90/100%), anomaly alerts, predictive alerts (2h warning), model switch detection. 24h dedup window. Dispatches to desktop, webhook, WebSocket.

💾 Storage

SQLite with WAL mode via aiosqlite. 5 tables: token_events, sessions, adapter_state, settings, anomalies. Composite index on (tool, timestamp) for fast queries.

🖥️ Frontend

React + TypeScript + Vite SPA. Real-time charts via WebSocket with client-side interpolation. Dark theme dashboard with heatmaps, timelines, and efficiency scores.

🔌 Integrations

MCP server (stdio) for Kiro with 3 tools. Auto-generates .kiro/steering/token-budget.md every 30 min. Shell hook for prompt integration.

Features

📈

Burn Rate Forecasting

24-hour token consumption forecast with 80% and 95% confidence intervals. Holt-Winters seasonality or Prophet for weekly patterns.

🚨

Anomaly Detection

IsolationForest on 6 hourly features detects unusual patterns: context loading bursts, extended conversations, usage spikes.

⚡

Efficiency Scoring

5-factor weighted score (0-100) per session: output/input ratio, cache hits, turn count, context growth, cost per token.

💰

Budget Projection

Monthly cost projection with what-if simulation. Model context size changes, model switches, and usage adjustments.

🔔

Smart Alerts

Threshold (50/75/90/100%), anomaly, and predictive alerts. Desktop notifications, webhooks, and WebSocket broadcast with 24h dedup.

👤

Behavioral Profiling

KMeans clustering classifies usage into archetypes: Morning Sprinter, Steady Coder, Night Owl. Detects weekly drift.

📊

Usage Heatmap

24×7 heatmap showing token consumption by day-of-week and hour. Identify peak usage patterns at a glance.

🔍

Waste Detection

Detects repeated context loading, excessive back-and-forth, and context bloat patterns. Generates actionable recommendations.

Supported Tools

Claude Code

Parses JSONL conversation logs from ~/.claude/projects/. Tracks byte offsets for incremental parsing. Zero-config discovery.

✓ Native Adapter

Kiro

MCP server integration via stdio transport. Token estimation using tiktoken cl100k_base. Auto-generates steering files.

⚡ MCP Integration

Custom Tools

Extend via entry-point plugins. Implement the ToolAdapter interface: discover(), get_log_paths(), parse_file().
